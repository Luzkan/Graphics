Logomotion Online (Turtle Graphic)

Your browser does not support the HTML5 canvas tag.

Pencil: Down

Available Commands

  • mv 'num' - moves the turtle
  • r 'num' - rotates right with chosen degree
  • l 'num' - rotates left with chosen degree
  • e ( 'code' ) 'num' - echoes command "num" times
  • c - changes color (available: red, blue , green, yellow, brown, black)
  • down - starts drawing
  • up - blocks drawing

Drawing Examples

  • Circle: up mv 240 l 90 mv 400 r 90 down e ( mv 3 r 1 ) 360
  • House: up mv 100 l 90 down e ( mv 250 r 90 ) 3 up l 270 mv 250 l 45 down c red mv 177 l 90 mv 177 l 45 up mv 40 l 90 mv 40 down c blue e ( mv 50 r 90 ) 4 up mv 125 down e ( mv 50 r 90 ) 4 up mv 85 r 90 mv 180 mv 30 r 90 mv 200 r 90 down c brown mv 100 r 90 mv 50 r 90 mv 100
